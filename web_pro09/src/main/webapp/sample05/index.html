DOM 객체 선택/접근

document.getElementById(id)

HTML 태그의 속성중 id 속성의 값으로 태그를 선택/접근 하기 위한 방법
id 로 접근하는 경우 가장 먼저 선택한 요소 1개만 반환한다.

document.getElementByTagName(tag_name)

HTML 태그 이름으로 선택/접근 하기 위한 방법
동일한 태그명에 해당하는 모든 요소를 HTMLCollection으로 반환한다.

document.getElementByName(name)

HTML 태그의 속성중 name 속성의 값으로 태그를 선택/접근 하기 위한 방법
동일한 name 속성 값에 해당하는 모든 요소를 HTMLCollection으로 반환한다.

document.getElementClassName(class_name)

HTML 태그의 속성중 class 속성의 값으로 태그를 선택/접근 하기 위한 방법
동일한 class 속성 값에 해당하는 모든 요소를 HTMLCollection으로 반환한다.

document.getSelector(css_selector)

HTML 태그의 속성중 class 속성의 값으로 태그를 선택/접근 하기 위한 방법
css 선택자를 사용하여 id, 태그명, class 등의 속성을 활용할 수 있다
가장 먼저 선택한 요소 1개만 반환한다.

document.getSelectorAll(css_selector)

HTML 태그의 속성중 class 속성의 값으로 태그를 선택/접근 하기 위한 방법
css 선택자를 사용하여 id, 태그명, class 등의 속성을 활용할 수 있다
동일한 css 선택자에 해당하는 모든 요소를 NodeList 으로 반환한다.

선택된 DOM 객체를 기준으로 다른 DOM 객체 선택/접근

DOM 을 활용하여 사용자가 입력한 URL 주소와 이름이 ol, li, a 태그로 만들어져 목록화 될 수 있게 한다.
번호를 입력하지 않으면 항상 마지막에 추가되게 하며, 번호를 입력한
경우에는 해당 번호의 다음 목록으로 추가될 수 있도록 한다.
유효하지 않은 번호의 경우에도 마지막에 추가되게 한다.
